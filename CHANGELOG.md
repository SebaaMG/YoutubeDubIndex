# Changelog

## v1.0.0

First official release.

- Added the native Windows catalog for browsing confirmed Spanish-dubbed YouTube videos.
- Added manual and automatic discovery flows with chunked progress reporting.
- Added a separate discovery worker process to keep the UI responsive during search and inspection.
- Added compact video cards, high-resolution thumbnails, and smoother paginated scrolling.
- Added filters for language, sort order, upload dates, dub type, favorites, and maximum duration.
- Added a curated reaction-oriented discovery pool and portable starter database support.
- Added a PyInstaller one-folder build flow for Windows.
